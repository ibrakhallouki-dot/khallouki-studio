# Khallouki Studio

Premium design portfolio & marketplace platform scaffold.

Stack: React + TypeScript + Vite + Tailwind + Express + tRPC + Supabase + Drizzle ORM

Quick start

1. Install dependencies (requires pnpm):

   pnpm install -w

2. Run the app (runs both web and server in parallel):

   pnpm run dev

Web dev server: http://localhost:5173
API server: http://localhost:4000

This scaffold provides a minimal starting point. Continue by implementing routes, database models, Supabase integration, and UI pages defined in the project spec.
